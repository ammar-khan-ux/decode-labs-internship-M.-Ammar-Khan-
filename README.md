# decode-labs-project-Password-strength
Week 1 Cyber Security Project by DecodeLabs
🔐 Password Strength Checker

📌 Project Overview

Password Strength Checker is a simple Cyber Security GUI application developed using Python and Tkinter. This tool checks whether a password is Weak, Medium, or Strong based on password complexity rules.

This project was created as part of the DecodeLabs Industrial Training Program 2026.


🚀 Features

* Check password strength instantly
* Detects:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Show/Hide password option
* Attractive GUI using Tkinter
* Beginner-friendly cyber security project

 🛠️ Technologies Used

Python
Tkinter GUI Library

 🔍 Password Strength Rules

| Strength  | Conditions                                                   |
| --------- | ------------------------------------------------------------ |
| Weak ❌    | Password length less than 6                                  |
| Medium ⚠️ | Contains lowercase letters and numbers                       |
| Strong 💪 | Contains uppercase letters, numbers, symbols, and length ≥ 8 |
